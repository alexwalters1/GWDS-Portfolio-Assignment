CARP Typographic Systems Coding UX Design

Axial

This typographic system is designed around an imaginary line, known as an axis. All content designed within this system falls either side of this axis; this can be both symmetrical and asymmetrical – furthermore, the axis does not need to be straight, it can be angled, zigzag etc.

An example of this in nature is a tree, this is due to the multiple branches coming from a central system.

Bilateral

Simply bilateral can be translated to centred. In a bilateral typographic system, all content is aligned upon the centre of an axis, like ‘centre text’ formatting in text editors like Microsoft Word.

Again, this system is not limited to an axis is the centre of the canvas, if text/content is aligned centrally over an axis it is considered bilateral.

Dilatational

Dilatational design demonstrates the use of type along circular paths. Rather than radiating from a point, text is instead formed around the curve of the focal point shape.

Using the previous tree analogy – rings of tree trunks are a good example of this design, if there was text present.

Grid

Most present in day-to-day life, for example in newspapers, all text and content fall into columns and rows.

This typographic system favours text above other content and as a result of this there is rarely any doubt of readability – some designers use this exclusively and is applicable to lots of design scenarios.

Modular

Using repeating structures to break up content, modular typographic systems work very well alongside grid layouts, but they are not reliant on one another.

These aforementioned structures can be anything, but shapes are most common, this design is suitable for already separated content like a music festival poster with multiple stages and acts.

Radial

Radial typographic systems justify content to radiate from a central point, like a car tyre.

This type of design is mostly seen on posters, promotional material as well as interactive websites, due to this typographic system promoting interesting and visually enticing designs.

Transitional

This typographic system is similar to dilatational but is not limited to purely circular paths.

It resembles a cross-section of sediment and is one of the most unique systems shown on this page, text flows to resemble a design you would find in nature.

Random

Probably the most self-explanatory system.

Random typographic systems have no order and usually look quite jumbled and confusing, however if executed right it can have a strong impact to the target audience.
